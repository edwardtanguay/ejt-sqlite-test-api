SQLite API

This API demonstrates using qtools to create an API that serves data from an SQLite database.

Routes:

  • http://localhost:3345/employees
  • http://localhost:3345/employees/4
  • http://localhost:3345/employees-by-city/London
  • http://localhost:3345/employees-by-birth-year/1963
  • http://localhost:3345/employees-search/french
  • http://localhost:3345/employee-territories-by-employee/2
